Linguagem HTML

A Linguagem HTML (linguagem de marcação de hipertexto) é a principal linguagem na programação web.

Ela serve como base para o desenvolvimento web e pode ser incrementada com estilos e executar scripts.

Folha de Estilo em Cascata

É a parte da programação que envolve o estilo e apresentação geral da página web.

Um estilo pode ser aplicado tanto por tags quanto por arquivos .css separados do HTML.

Javascript

Linguagem de programação agregada na página HTML que envonve a lógica de funcionamento de scripts dentro da página.

Envolve toda a lógica de programação da página e possui uma sintaxe bem específica.
